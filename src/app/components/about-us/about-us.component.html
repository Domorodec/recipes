Where did it begin?

In 2019 bank world become crazy about "agile" access to everything where this could be implemented and banks have started to transform. This huge change brought a lot of changes and one of them was that it put on table new offers regarding job change.
This means...agile worl allowes you to jump to another really different position where you think could be better place for you to work and express yourself. So my dream has always was to become DEVELOPER or PROGRAMMER so I have applied to web development position and before interview started to learn basic about programming starting with JAVA.
Well after couple of months I was being given by chance my dream to become true and my new contract held job title - DEVELOPER - "jupi ja jej" .. ☺

What I have learned?

  • Working within team of senior developers isn't always easy, specially for newbie junior developer but it helps you become more independent, forces you to ask, ask and ask and teaches you to be patient and tolerant.
  • Git is not place for chatting or expressing you ideas, so your commit messages should be up to date with short but relevant info about your changes in your branch.
  • Camel case is not the same as snake case or even pascal case so naming convention style should be kept as standard.
  • Remember that null value or empty string should not be welcome friend in each API
  • Debug, debug and debug and try to understand logic of application you are working on.
